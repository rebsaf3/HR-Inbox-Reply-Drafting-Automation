# Contributing
This repo is a public case study. Contributions are welcome if they improve clarity, safety, and usefulness without adding proprietary details.

## Contribution rules
1. Do not add tenant names, company names, mailbox addresses, tokens, connector ids, or internal urls
2. Do not add new claims about HR policy content
3. Keep language tool neutral and vendor neutral
4. Favor deterministic patterns and testable behaviors

## Change process
1. Open an issue using the Content Update form
2. Propose edits with a short rationale and the impacted docs
3. Submit a pull request referencing the issue
