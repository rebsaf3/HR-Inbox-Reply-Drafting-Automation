# Public Repo Redaction Checklist
Use this list before publishing.

1. Remove tenant names and tenant ids
2. Remove company names and brand names
3. Remove mailbox addresses and group ids
4. Remove connector names that reveal internal systems
5. Remove tokens, secrets, keys, and connection references
6. Remove internal urls, share links, and document ids
7. Remove employee names and personal data
8. Remove screenshots that show private data
9. Replace real dates with generic dates when possible
10. Confirm examples do not include real policy statements

If a detail is not required to explain the design, remove it.
